# MyProgrammingPortfolio

This repository contains my program for the C++ Calculator assignment.

## About the Program
This project includes a simple calculator written in C++ which performs:
- Addition
- Subtraction
- Multiplication
- Division

## How to Run
1. Open any C++ compiler (OnlineGDB / VS Code / CodeBlocks).
2. Open the file `calculator.cpp`.
3. Run the program.
4. Enter:
   - First number
   - Operator (+, -, *, /)
   - Second number
